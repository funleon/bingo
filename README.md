# 🎰 賓果遊戲系統

一個純前端的互動式賓果遊戲系統，採用 HTML、CSS 和 JavaScript 開發。具有逼真的球籠滾動動畫、絢麗的花火特效，以及靈活的遊戲配置功能。

## ✨ 特色功能

### 🎯 核心功能
- **球籠動畫** - 球在籠內獨立散開滾動，模擬真實的抽獎機効果
- **花火特效** - 抽取球時爆發彩色粒子特效，增強視覺震撼
- **靈活配置** - 可自訂號碼區間（如 1-90、1-50 等）
- **自由設置** - 每次可設定抽取 1-10 顆球
- **實時統計** - 顯示已抽球數和配置號碼數

### 🎮 使用者界面
- **左側面板** - 遊戲設置和統計資訊
- **中央區域** - 球籠動畫、抽球按鈕和本次抽取顯示
- **右側面板** - 完整的已抽獎球列表（歷史記錄）

## 🚀 快速開始

### 部署方式

#### 方式 1：本地開啟（推薦）
1. 將項目文件夾放在本地硬碟
2. 用瀏覽器直接打開 `index.html`
3. 無需任何伺服器或依賴

#### 方式 2：簡單 HTTP 伺服器
如果在某些瀏覽器中遇到 CORS 問題，可啟動簡單伺服器：

**使用 Python 3：**
```bash
cd d:\Projects\bingo
python -m http.server 8000
```

**使用 Python 2：**
```bash
cd d:\Projects\bingo
python -m SimpleHTTPServer 8000
```

**使用 Node.js：**
```bash
cd d:\Projects\bingo
npx http-server
```

然後用瀏覽器訪問 `http://localhost:8000`（或相應的伺服器地址）

#### 方式 3：部署到網站伺服器
將整個 `bingo` 文件夾上傳至你的網站空間：
```
your-website.com/bingo/
```

## 📖 操作指南

### 初始設置

1. **打開頁面** - 在瀏覽器中打開 `index.html`

2. **配置遊戲參數**（左側面板）
   - **最小號碼** - 設置抽獎號碼的最小值（預設：1）
   - **最大號碼** - 設置抽獎號碼的最大值（預設：90）
   - **一次抽幾顆球** - 設置每次抽取的球數量（預設：1，最多 10 顆）

3. **套用設置** - 點擊「套用設置」按鈕確認配置
   - 配置號碼數會自動更新
   - 球籠會重新生成球颐

### 開始抽獎

1. **確認設置** - 確保所有參數符合要求

2. **點擊抽球按鈕** - 中央的大紅圓形按鈕
   - 按鈕會暫時禁用（灰色），直到抽取完成
   - 球籠內的球會滾動
   
3. **查看結果**
   - **本次抽取** - 中央顯示本次抽取的球號碼
   - **花火特效** - 每顆球會產生彩色粒子特效
   - **已抽獎球列表** - 右側會逐個添加球號碼

### 遊戲統計

左側面板會實時顯示：
- **已抽球數** - 目前已抽取的球數
- **配置號碼數** - 總共可以抽取的球數
- **配置號碼範圍** - 當前設置的號碼區間

### 重置遊戲

點擊「重置遊戲」按鈕可以：
- 清空所有已抽球記錄
- 清除以抽獎球列表
- 清空本次抽取顯示
- 重新生成球籠

## 🎨 動畫效果說明

### 球籠滾動
- 球在半徑 350px 的圓形籠內隨機分散
- 每個球以 4 秒周期滾動，路徑隨機
- 球與球之間不會重疊（獨立動畫）
- 球籠背景以 8 秒速度緩慢旋轉

### 花火特效
- 每抽取一顆球，在隨機位置爆發花火
- 花火由 30 顆彩色粒子組成（紅、青、黃、粉、綠）
- 粒子向四周飛散，1.2 秒後逐漸淡出消失
- 支持多顆球同時產生花火

### 抽球過程
1. 點擊抽球按鈕
2. 球籠開始滾動（視覺反饋）
3. 逐個抽取球（每 300ms 一顆）
4. 每抽取一顆球：
   - 在中央顯示該球號碼
   - 產生花火特效
   - 添加到右側列表

## 🛠️ 自訂和擴展

### 修改配色
編輯 [style.css](assets/css/style.css) 中的 CSS 變數：

```css
:root {
    --primary-color: #ff6b6b;      /* 主要紅色 */
    --secondary-color: #4ecdc4;    /* 次要青色 */
    --accent-color: #ffd93d;       /* 強調黃色 */
    --dark-bg: #2c3e50;            /* 深色背景 */
    --light-bg: #ecf0f1;           /* 淺色背景 */
}
```

### 修改動畫速度
在 [style.css](assets/css/style.css) 中調整動畫時間：

```css
@keyframes ballRoll {
    /* 修改 4s 為其他值來改變球滾動速度 */
}

@keyframes cageSlowRotate {
    /* 修改 8s 為其他值來改變籠旋轉速度 */
}
```

### 修改遊戲邏輯
編輯 [bingo.js](assets/js/bingo.js) 中的 `BingoGame` 類：

```javascript
// 修改預設設置
this.minNumber = 1;      // 預設最小號碼
this.maxNumber = 90;     // 預設最大號碼
this.ballCount = 1;      // 預設每次抽幾顆
```

## 📋 常見問題

### Q: 為什麼球籠中球的數量很多時顯示不全？
A: 這是正常的。球籠的容量是有限的，當球籠太滿時，球會相互覆蓋。這在某種程度上也模擬了真實情況。

### Q: 可以修改花火的顏色和數量嗎？
A: 可以。編輯 [bingo.js](assets/js/bingo.js) 中的 `createFireworks()` 方法：
```javascript
const colors = ['#ff6b6b', '#4ecdc4', '#ffd93d', '#ff8787', '#44b3aa'];
const particleCount = 30;  // 修改為其他數字
```

### Q: 可以讓球籠轉得更快或更慢嗎？
A: 可以。在 [style.css](assets/css/style.css) 中修改動畫時間。

### Q: 支持離線使用嗎？
A: 是的，完全支持。無需網絡連接，可以離線正常使用。
